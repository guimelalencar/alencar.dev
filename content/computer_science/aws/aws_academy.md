# AWS Academy

A AWS Academy fornece às instituições de ensino superior um currículo de computação em nuvem gratuito e pronto para ensinar, que prepara os alunos para buscar certificações reconhecidas no setor e tarefas na nuvem em demanda.  
Nosso currículo ajuda os educadores a permanecerem na vanguarda da inovação na Nuvem AWS, equipando os alunos com habilidades necessárias para atuar em um dos setores que mais crescem.

---

## Cursos

### Foundational

- **[[aws_academy_cloud_foundation|AWS Academy Cloud Foundation]]**
  Curso introdutório destinado a alunos que buscam compreensão geral sobre conceitos de computação em nuvem. Abrange conceitos de nuvem, produtos essenciais da AWS, segurança, arquitetura, preço e suporte.

- **AWS Academy Generative AI Foundations**  
  Introdução à inteligência artificial com ênfase em IA generativa. O curso fornece ampla compreensão de IA e machine learning (ML) e explora conceitos, capacidades, casos de uso e serviços AWS relacionados à IA generativa.

- **AWS Academy Machine Learning Foundations**  
  Curso introdutório sobre conceitos e terminologia de IA e ML. Os alunos aprenderão a selecionar e aplicar serviços de ML, rotular, criar, treinar e implantar modelos personalizados.  
  Duração: aproximadamente 20 horas (palestras, laboratórios e projetos).

- **AWS Academy Data Center Technician (DCT)**  
  Desenvolve conhecimentos técnicos em operações de data center. Requer conhecimento básico de engenharia mecânica e elétrica.

- **AWS Academy Engineering Operations Technicians (EOT)**  
  Visão geral de conceitos e habilidades para técnicos de operações de engenharia em data centers, incluindo infraestrutura física, fundamentos elétricos, baterias e geradores.

- **AWS Academy Cloud Security Foundations**  
  Introdução aos princípios e serviços de segurança cibernética para nuvem. Inclui demonstrações práticas, guias de instrução e cenários do mundo real.

---

### Associate

- **AWS Academy Cloud Architecting**  
  Curso intermediário sobre infraestrutura de TI na AWS, preparando para a certificação AWS Certified Solutions Architect – Associate.  
  Duração: aproximadamente 40 horas (palestras, laboratórios e projetos).  
  **Pré-requisito:** AWS Academy Cloud Foundations ou equivalente.

- **AWS Academy Cloud Developing**  
  Curso intermediário focado em desenvolvimento em nuvem e preparação para AWS Certified Developer - Associate. Desenvolve habilidades em AWS SDK e melhores práticas para construção e implantação de aplicativos.  
  Duração: aproximadamente 40 horas.  
  **Pré-requisito:** AWS Academy Cloud Foundations ou equivalente.

- **AWS Academy Cloud Operations**  
  Prepara para DevOps, suporte e operações em nuvem de nível de entrada, e para AWS Certified SysOps Administrator – Associate. Inclui estudos de caso, demonstrações e laboratórios práticos.  
  Duração: aproximadamente 40 horas.  
  **Pré-requisito:** AWS Academy Cloud Foundations ou equivalente.

- **AWS Academy Data Engineering**  
  Ensina coleta, armazenamento, preparação, análise e visualização de dados para analytics e machine learning, com exploração de casos de uso reais.

- **Machine Learning para Processamento de Linguagem Natural (PLN)**  
  Curso intermediário voltado a estudantes que buscam carreiras com ML. Ensina termos de PLN, aplicações em negócios e soluções para diferentes problemas usando PLN.

### Laboratórios

- **Laboratório de alunos da AWS Academy**  
  Ambientes práticos de longa duração onde educadores podem criar atividades e convidar alunos para experiência com produtos AWS.  
  Funcionalidades:  
  - Criar sala de aula  
  - Convidar alunos  
  - Atribuir projetos  
  - Visualizar área de trabalho dos alunos  
  - Análises de tempo e custo  
  - Acesso a mais de 100 produtos AWS  
  - Disponível para educadores que completaram _Getting Started with AWS Academy_

---

### Projetos de Laboratório

- **AWS Academy Cloud Data Pipeline Builder**  
  Criação de pipeline de analytics usando AWS, baseado em dados de sequência de cliques de site, seguindo o AWS Well-Architected Framework.

- **AWS Academy Cloud Web Application Builder**  
  Projeto para projetar e implantar uma aplicação web com banco de dados na nuvem, seguindo princípios de alta disponibilidade, escalabilidade, desempenho e segurança.

- **AWS Academy Cloud Security Builder**  
  Proteção de recursos na AWS. Projeto dividido em quatro fases independentes, seguindo o AWS Well-Architected Framework e princípio do privilégio mínimo.

- **AWS Academy Microservices and CICD Pipeline Builder**  
  Criação de solução de microsserviços e pipeline CI/CD. Inclui migração de aplicação monolítica para arquitetura baseada em contêiner, permitindo escalabilidade independente dos componentes.
